Column — Why More Public Bidders Matter

公募案件における応札不足の構造と
応札促進の社会的意義

一者応札は、直ちに不正や手続違反を意味するものではありません。しかし、競争可能性が低い状態が常態化すれば、価格発見、提案比較、品質改善、新規参入、供給の強靱性が損なわれます。問題を「企業が応募しないから」と片づけず、応札を阻む構造と、その解消が公共調達にもたらす価値を整理します。

公共調達は、公告を公開し、参加資格を満たす事業者に門戸を開けば、それだけで十分に競争的になるわけではありません。

案件を発見し、資料を取得し、自社の資格・実績・人員を照合し、契約条件と採算を読み、提案書と証明資料を整える。応札者は、受注できるか分からない段階で相当の調査・判断・作成コストを負担します。専門性の高い調査研究、IT、運用保守、データ利用を伴う案件では、この負担はさらに大きくなります。

したがって、応札不足は単なる「周知不足」ではありません。発注条件、情報設計、既存事業者への依存、応募準備コスト、契約リスク、社内意思決定の遅さが重なって生じる、市場設計上の問題です。

見るべき指標は、公告が形式上公開されているかではなく、
複数の適格事業者が「合理的に参加を検討できる状態」にあるかです。

The issue

一者応札で失われるのは「安さ」だけではない

一者応札でも、手続が適法で、仕様を満たし、価格が予定価格内であれば契約は成立します。特定技術、知的財産、互換性、緊急性等により、供給者が事実上一者に限られる場合もあります。

しかし、競争契約の形式を採りながら応札者が一者にとどまる状態が反復するなら、発注者は次の機能を十分に利用できません。

価格発見と条件規律

  • 複数の市場価格を比較できない
  • 工数・単価・リスク配分の妥当性を検証しにくい
  • 既存事業者に条件改善を促す圧力が弱い

提案比較と学習

  • 異なる技術方式・実施体制を比較できない
  • 仕様自体の過不足に気づきにくい
  • 市場から新しい解決方法を取り込めない

供給の強靱性

  • 特定事業者への依存が固定化する
  • 撤退・障害・値上げ時の代替が乏しい
  • 運用知識やデータが閉じやすい

正統性と説明可能性

  • 「なぜその価格・提案なのか」の説明材料が減る
  • 形式的競争と実質的競争の乖離が生じる
  • 市場参加者から見た公平性への信頼が弱まる
95.4%

会計検査院が分析した裁判所の情報システム関連競争契約では、一者応札の平均落札率。複数応札は69.0％でした。対象年度・分野が限定された事例であり、全公共調達への単純な一般化はできません。[1]

−15%

フィンランドの2017～2025年の公共調達データを用いた2026年の研究では、一者ではなく二者以上の応札がある案件は、調達費用が15％低いことと関連しました。[2]

−9%

同研究では、一者応札案件に一者だけ追加で参加した場合の推計削減幅は9％。因果効果の一般値ではなく、同国データに基づく推計として読む必要があります。[2]

数字の読み方

落札率や応札者数だけで個別契約の適否は判断できません。重要なのは、一者応札を異常値として機械的に排除することではなく、競争可能性が低下した理由を案件類型ごとに検証することです。

Two-sided friction

応札不足は、発注側と事業者側の摩擦が重なって生じる

発注者は「公告したが応募が来なかった」と見ます。事業者は「案件は見えたが、応募を合理化できなかった」と判断します。この認識差を埋めない限り、周知先を増やすだけでは応札者は増えません。

事業者側の摩擦

  • 発見コスト：機関ごとに公告ページ、検索方法、資料取得方法が異なる
  • 判定コスト：資格、等級、地域、実績、人員、認証、再委託条件を短期間で照合する必要がある
  • 提案コスト：受注確率が不明なまま、技術提案、見積り、工程、体制、証明資料を作成する
  • 機会費用：提案要員を既存顧客や民間営業から外す必要がある
  • 契約リスク：検収、知財、損害賠償、実費精算、支払時期、監査対応の負担を読みにくい
  • 実行不確実性：データ、既存システム、関係者、現場条件へのアクセスが契約後まで確定しない

発注設計側の摩擦

  • 過剰要件：目的達成に必須でない実績、資格、認証、専任体制を求める
  • 準備期間不足：公告から締切までが短く、社内承認や協力会社調整が間に合わない
  • 既存事業者優位：運用知識、設計書、データ、著作権、交換部品等へのアクセス差がある
  • 仕様の固定化：特定製品・既存方式に事実上適合する提案しか成立しない
  • 採算の不透明さ：予定工数、責任範囲、価格改定、追加作業の扱いが読みづらい
  • 情報分散：公告、仕様、質問回答、訂正、様式が別ページやメール請求に分かれる

会計検査院は、情報システムの改修・保守では、システムへの習熟費用、プログラム解析の困難さ、要求仕様を満たせないリスクが新規事業者の参加を妨げ得ると整理しています。設計書等の閲覧期間を長くすることや、広く参加できる仕様への見直しが改善策として挙げられています。[1]

応札不足が続くと、その結果自体が次回調達の参入障壁を強める。

Contestability

目標は「応札数」ではなく、競争可能性を高めること

応札者が多ければ常に良いわけではありません。要件を満たさない提案、履行能力のない低価格入札、形式的な相見積りを増やしても、調達の質は向上しません。

応札促進で目指すべきなのは、適格で実行可能な複数事業者が、過度な負担なく参加を判断できる状態です。これは、次の三つに分解できます。

01 / DISCOVERABLE

見つけられる

対象市場の事業者に公告が届き、更新・質問回答・締切を追跡できる。情報が機関ごとに孤立しない。

02 / ASSESSABLE

判断できる

資格、実績、予算、評価基準、契約条件、利用可能なデータが明確で、Go／No-Goを早期に判定できる。

03 / EXECUTABLE

履行できる

準備・履行期間が現実的で、既存事業者だけが持つ情報への依存が抑えられ、責任と対価が均衡する。

形式上は誰でも参加できるが、実務上は一社しか提案を成立させられない。問題となるのは、この状態です。

Public value

応札促進が持つ五つの社会的意義

1. 財政支出の効率化

VALUE FOR MONEY

複数の価格・工数・実施方法を比較できれば、予定価格に近い一案を受け入れるだけでなく、価格と品質の均衡を検証できます。競争は単純な値下げ圧力ではなく、費用内訳とリスク配分を市場に問い直す手段です。

2. 品質・技術選択肢の拡大

QUALITY

調査、IT、コンサルティング、研究開発支援では、同じ目的に複数の実現方法があります。比較可能な提案が増えることで、発注者は価格だけでなく、方法論、体制、保守性、説明可能性、将来拡張性を選択できます。

3. 中小企業・新規事業者の市場参入

MARKET ACCESS

公共調達は、新規顧客への入口であると同時に、その後の提案で使える実績を形成する市場です。最初の一件へのアクセスを改善すれば、市場の固定化を緩め、地域企業・専門企業・新設企業の成長機会を広げられます。2026年度の国の基本方針も、中小企業・小規模事業者向け契約目標61％、新規中小企業者向け3％以上を掲げています。[5]

4. 供給網と行政サービスの強靱化

RESILIENCE

特定事業者への依存を減らし、代替可能な事業者と技術方式を育てることは、障害、撤退、価格上昇、災害、サイバーインシデント時の継続性に寄与します。競争政策と事業継続は別の課題ではありません。

5. 調達への信頼と政策学習

LEGITIMACY

複数提案を比較した記録は、採択理由の説明可能性を高めます。また、応募が少なかった理由、質問、辞退要因を蓄積すれば、次回の仕様、期間、契約条件を改善する学習データになります。

Buyer actions

発注者が改善できること

応札促進は、広告費を増やすことだけではありません。参加判断と履行に必要な摩擦を、調達目的を損なわない範囲で減らすことです。

要件・仕様

  • 参加資格と過去実績を必要最小限にする
  • 特定製品・既存方式への不要な依存を除く
  • 同等品・代替方式・共同提案の可否を明示する
  • 大規模案件を適切に分割し、中小企業が担える単位を作る

時間・情報

  • 提案書の量に見合う公告・準備期間を確保する
  • 質問回答、訂正、様式を一か所で追跡可能にする
  • 既存設計書、データ定義、運用条件の閲覧機会を設ける
  • 予告、情報提供依頼、意見招請で市場対話を前倒しする

契約・採算

  • 成果範囲、検収、追加作業、知財、賠償責任を明確にする
  • 物価・人件費変動に対する協議・価格改定の扱いを示す
  • 過度な無償提案や応募時成果物を求めない
  • 履行期間と支払条件を実行可能な水準にする

検証・改善

  • 一者応札案件を類型別にレビューする
  • 資料請求後の不参加・辞退理由を匿名で収集する
  • 応札者数だけでなく、有効提案数と市場参加者数を追う
  • 仕様変更後の応札数、価格、品質を継続比較する

京都大学が公表する改善方策も、参加資格を必要最小限にすること、適切な履行期間、広い周知、提案書類の多い調達で十分な準備期間を確保することを挙げています。[3] また、会計検査院の分析では、建設コンサルタント業務等で公告期間が長い区分ほど一者応札率が低い傾向が確認されています。[4]

Supplier actions

事業者側にも「応札できる状態」の整備が必要

発注条件が改善されても、事業者側に資格・実績・判断プロセスが整っていなければ、締切までに応札へ移れません。公募営業では、案件を見つけてから毎回ゼロから確認する運用自体が参入障壁になります。

✓

資格レディネス：等級、営業品目、地域資格、電子調達、認証、納税証明等を平時に整理する。

✓

実績の構造化：案件名、発注者、期間、担当範囲、成果、証明資料、開示可否を台帳化する。

✓

人員・協力先の可視化：必要専門性を誰が担えるか、再委託・共同提案が可能かを事前に確認する。

✓

Go／No-Go基準：親和性、フィジビリティ、収益性、戦略価値、提案工数、契約リスクを短時間で判定する。

✓

提案資産の再利用：体制図、品質管理、リスク管理、類似実績、見積前提を検証済み部品として蓄積する。

✓

辞退理由の記録：資格不足、実績不足、人員不足、採算、期間、契約条件を分け、次の改善につなげる。

Shared responsibility

応札促進は、発注者だけの課題でも、営業支援だけの課題でもない

発注者が担う市場設計

  • 必要最小限で中立的な要件
  • 十分な準備・履行期間
  • 比較可能でアクセス可能な情報
  • 対価と責任が均衡した契約
×

事業者が担う参加能力

  • 資格・実績・人員の平時整備
  • 迅速な適格性・採算判断
  • 証明可能な提案と履行体制
  • 実績と判断履歴の再利用

一方だけを改善しても、競争可能性は十分に高まりません。発注者が参加しやすい市場を設計し、事業者が参加できる組織能力を持つ。その接点に、公告情報の標準化、調達インテリジェンス、資格・実績データ、事前市場対話が位置します。

EUの復興・強靱化計画の枠組みでは、ある加盟国が、EU基準額の上下を問わず一者応札手続の比率を15％未満まで引き下げ、その水準を維持することを明示的な改革目標に掲げています。これは「一者応札は違法」という意味ではなく、競争性・透明性・効率性を政策目標として測定する考え方を示しています。[6]

PSPA-Screen

PSPA-Screenが担うのは、応札前の取引コスト削減です

PSPA-Screenは、発注者の評価や落札結果を操作するものではありません。また、無差別な大量応募や自動送信を目的としません。

対象とするのは、適格な企業が案件を見落とさず、応募可能性を早期に評価し、根拠のあるGo／No-Go判断を行い、提案に必要な事実と証拠を再利用できる状態を作ることです。

案件を見つける

分散した発注機関の公告を取得し、対象外案件を一次トリアージして、検討可能性のある案件に集中します。

応募可能性を判断する

資格、実績、人員、予算、地域、契約条件を照合し、親和性とフィジビリティを分けて評価します。

根拠を揃える

ファクト台帳と証明資料を対応させ、未確認事項、開示可否、追加ヒアリング事項を明示します。

判断を次に残す

応札・辞退・不採択・採択の履歴と理由を蓄積し、次回の判断速度と提案品質を高めます。

応札数を機械的に増やすのではなく、
適格な事業者が、参加すべき案件に参加できる確率を高める。

PSPA-Screenは、送信・自動応募の経路を構造的に持ちません。最終的な応札判断と提出操作は、利用企業が行います。

Conclusion

競争性は、公告の瞬間ではなく、その前後の設計で決まる

一者応札は結果です。その背景には、要件、期間、仕様、契約、情報アクセス、既存システムへの依存、事業者の資格・実績管理、提案コストといった複数の要因があります。

だからこそ、応札促進は「もっと企業に応募してもらう」という広報施策にとどまりません。発注者にとっては、比較可能な市場を設計すること。事業者にとっては、参加判断と提案を反復可能な業務プロセスにすることです。

適格な応札者が増えれば、価格だけでなく、品質、技術選択肢、新規参入、供給の強靱性、説明可能性が改善します。公共調達の競争性を高めることは、単なる調達事務の効率化ではなく、公費から得られる社会的価値を高めるための市場基盤整備です。
